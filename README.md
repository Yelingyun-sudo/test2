# Website Analytics

## 模块
- `backend/`：Python 核心逻辑 + FastAPI，详见 `backend/README.md`
- `frontend/`：Next.js（App Router），详见 `frontend/README.md`

## 常用路径
- 指令模板：`backend/instructions/`
- 任务资源：`backend/resources/`
- 运行日志：`backend/logs/`

## 开发入口
- 后端：`cd backend` 后按 README 安装依赖、运行 CLI 或 API。
- 前端：`cd frontend` 后按 README 安装依赖、运行开发服务器。
